# Установка приложения на телефон

## 1. Узнайте IP вашего компьютера

Телефон и ПК должны быть в одной Wi‑Fi сети.

**Windows (PowerShell):**
```powershell
ipconfig | findstr "IPv4"
```

Ищите строку вида `192.168.1.100` (или `192.168.0.x`).

**Или:** Параметры → Сеть и Интернет → Wi‑Fi → Свойства → IPv4-адрес.

---

## 2. Пропишите IP в приложении

Откройте `lib/core/config/api_config.dart` и замените `192.168.1.100` на IP вашего ПК:

```dart
const String _apiHost = '192.168.1.100'; // ваш IP
```

---

## 3. Запустите backend на ПК

```bash
cd backend
npm start
```

Сервер должен слушать на `http://0.0.0.0:3000` (все интерфейсы).

---

## 4. Соберите APK

```bash
cd c:\myJts\my_new_project
flutter build apk
```

APK будет в `build/app/outputs/flutter-apk/app-release.apk`.

---

## 5. Установите APK на телефон

### Вариант А: USB‑кабель

1. Подключите телефон к ПК.
2. Включите «Отладка по USB» в настройках разработчика.
3. Выполните:
```bash
flutter install
```

### Вариант Б: Без кабеля

1. Скопируйте `app-release.apk` на телефон (облако, мессенджер, email).
2. На телефоне откройте файл и установите.
3. Разрешите установку из неизвестных источников, если система попросит.

---

## 6. Проверка

1. Backend запущен на ПК.
2. Телефон и ПК в одной Wi‑Fi.
3. Установите приложение.
4. Войдите: admin/admin123 или ivanov/user123.

---

## Возможные проблемы

| Проблема | Решение |
|----------|---------|
| «Требуется авторизация» | Неверный IP или backend не запущен |
| «Нет подключения» | Телефон и ПК в разных сетях? Проверьте IP |
| Файрвол блокирует | Разрешите входящие на порт 3000 |
